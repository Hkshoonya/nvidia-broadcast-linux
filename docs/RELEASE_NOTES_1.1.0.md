# NV Broadcast 1.1.0

Release date: 2026-03-27

## Highlights

- Added a collapsible Meeting Assistant sidebar with live on-device transcript updates.
- Added local meeting session history inside the app with automatic 7-day retention cleanup.
- Added mixed meeting audio capture so meeting notes and transcripts can follow both sides of the conversation.
- Added non-blocking optional runtime installs for CUDA, TensorRT, and Whisper with visible progress in the app.
- Improved first-run setup with clearer mode guidance, explanations, and skip/install flows.

## Release posture

- `v1.0.2` remains the staged stable baseline that was published first across package channels.
- `1.1.0` is the feature upgrade on top of that baseline.
- `DocZeus` remains the recommended default quality mode.
- The learned neural matte refiner remains experimental and disabled by default.
